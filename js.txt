this is bound at call time 
es6 arrow function binds this statically 
a constructor function is a function which defines properties inside function body : this.name = name 
calling new on an constructor fn has 4 effects: 1 create new object, 2 assign properties to fn object, 3 link the prototype, 4 return object 
a constructor is like a template, when called the passed arguments are applied to the properties 
NaN is a number, when an operation with a number is done which does not result in a new number: 5 * 'Word' = NaN
