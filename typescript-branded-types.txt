Typescript Type Branding
-------------------------
With ts-brand , you can achieve nominal typing by leveraging a technique that is called 
"type branding" in the TypeScript community. 

intersecting a base type with a object type with a non-existent property
-------------------------------------------------------------------------
Type branding works by intersecting a base type with a object type with a non-existent property. 
It is closely related in principal and usage to Flow's opaque type aliases
