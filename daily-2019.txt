Angular is best used in Enterprise Scenarios.

React is a frontend library in helping you to create a Web Frontend. May more people engage 
in React because it is easier to learn and not so demanding like Angular.

VueJs is in the concepts similar to Angular, but easier to learn, thats why most people want to learn it.
The community is not so big like in React or Angular.
